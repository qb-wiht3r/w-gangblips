# W-GangBlips

W-GangBlips is a lightweight and efficient gang tracking system for FiveM servers, designed to improve communication and coordination between gang members.

The script displays real-time map blips for gang members, allowing players to easily locate their allies and work together more effectively during activities, events, or conflicts.

### Features

* Real-time gang member blips
* Only visible to players within the same gang
* Automatic blip updates based on player position
* Customizable blip colors, sprites, and settings
* Optimized for performance
* Easy configuration and integration
* Seamless QBCore compatibility

### Dependencies

* qb-core

W-GangBlips enhances gang gameplay by providing better awareness, coordination, and teamwork while maintaining a lightweight and optimized experience.
